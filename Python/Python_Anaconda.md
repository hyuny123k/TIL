## 1. Anaconda 기본 명령어(Basic Command)

### (1) Managing conda

- **conda version check**

```pseudocode
#  아나콘다 버전 확인
	conda --version
```

: 설치한 버전을 표시해준다. 따로 Anaconda 디렉토리로 이동할 필요가 없다.



```
#  아나콘다 업데이트
	conda update conda
```

: Conda의 버전을 최신버전과 비교한 다음 설치여부를 묻고 업데이트한다.
